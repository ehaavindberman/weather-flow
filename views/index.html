Weather Flow

This is a work in progress and meant to be similar to the Fresh Air iPhone app which is no longer updated.

I am using the climacell API for daily and hourly forecasts, google maps places API for location search, weather icons from Erik Flowers, and a Node.js backend via repl.it.

There are still many things to do, but I think this is a working minimum product. Some of the things that I will update in the future include:

- Updating background color based on time of day and cloud cover (I'm already bringing in those varibales from Climacell's API, but have not implented the background coloring)

- Making colors gradient instead of flat

- Allowing users to swtich between Farenheit and Celsius, between temperature, wind speed, and feels like, and between precipitation, humidity, and cloud cover.

- Including high and low termperatures (or wind speed or feels like) for the day at the top with the temperature

- Showing where the user is on the graph at all times with a dot at the data value

- Highlighting the precipitation (or humidity or cloud cover) percentage in the circle in the top right

- Showing moon phases and sunrise and sunset times via dotted lines in the graph

- Stopping day break lines at the top of the temperature data instead of at the top of the canvas

- Interpolate between hourly data so that users can see minute-by-minute statistics

- Break away from the mobile-first design and find a way to show and interact with the weather cleanly and beautifully on desktop

So, as you can see there is still much to do, but I am excited to keep improving and this has definitely been a fun project so far!
